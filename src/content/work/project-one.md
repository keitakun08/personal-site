Project Alpha is a real-time analytics dashboard designed for data-driven teams. Built with React and TypeScript, it leverages D3.js for interactive visualizations and WebSocket connections for live data updates.

## Key Features

- Real-time data streaming via WebSocket
- Custom D3.js visualizations
- Role-based access control
- Export to CSV and PDF

## Tech Stack

The project uses React 18, TypeScript, D3.js, Node.js backend with PostgreSQL.
